## Detailed description (16,000 chars max)
## Hover Translate とは

Hover Translate は、英語と日本語の Web ページを読むときの中断を減らすための Chrome 拡張機能です。テキストを別のアプリに貼り付けたり、ページ全体を翻訳したりする代わりに、段落へホバーするだけで、その場にツールチップで翻訳結果を表示できます。

この拡張機能は用途を意図的に絞っています。英語⇄日本語の段落単位翻訳に限定し、翻訳エンジンにはユーザー自身の DeepL Free API キーを使い、UI も最小限にしています。設定用のポップアップ、結果表示用のフローティングツールチップ、そして ON/OFF を切り替えるショートカットだけです。

ドキュメント、記事、ブログ、技術資料などを英日で読み比べる場面で、邪魔になりにくい軽量な翻訳ツールとして使えるように設計しています。

## 主な機能

- 段落、リスト項目、見出し、表セルなどのブロック要素にホバーして、その場で翻訳
- 英語か日本語かを自動判定し、翻訳方向を自動で選択
- DeepL Free API を使用
- 300 ms のホバーディレイ後に、フローティングツールチップで結果を表示
- エラー時は赤いボーダー付きのツールチップで明確に表示
- ポップアップ UI は Setup / Translate / Quota / Limits の 4 セクション構成
- DeepL の使用量をポップアップ内に表示し、警告・危険状態も可視化
- 1 リクエストあたりの最大文字数を 500〜5000 の範囲で設定可能
- Service Worker 内の 100 件 LRU キャッシュで同じテキストの再翻訳を回避
- グローバルショートカットで翻訳機能の ON/OFF を切り替え可能

## 使い方

1. 拡張機能のポップアップを開き、DeepL Free API キーを保存します。
2. ポップアップで翻訳を ON にします。
3. 英文または日本語文を含む Web ページを開きます。
4. 段落サイズのテキストブロックにホバーすると、ツールチップで翻訳が表示されます。

## DeepL Free API キーが必要です

Hover Translate には共有の翻訳アカウントは含まれていません。翻訳機能を使うには、ユーザー自身で DeepL API Free アカウントを作成し、取得した API キーをポップアップに貼り付ける必要があります。

DeepL API Free では、現時点で月 500,000 文字まで無料で利用できます。拡張機能にはクォータ表示と文字数制限設定があるため、ポップアップから使用量を管理できます。

## プライバシー

ホバーしたテキストは、翻訳のために DeepL Free API にのみ送信されます。アナリティクス、トラッキング、クラッシュレポート、その他の不要なテレメトリは含まれていません。

DeepL API キーは `chrome.storage.local` にローカル保存されます。拡張機能は、認証付きの API リクエストを DeepL に送る場合を除き、キーを他の場所へ送信しません。

## キーボードショートカット

既定のショートカットは `Alt+Shift+T` です。現在のタブで翻訳機能を ON/OFF し、小さな状態トーストを表示します。ショートカットは Chrome の拡張機能ショートカット設定から変更できます。
