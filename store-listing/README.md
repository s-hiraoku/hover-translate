# Store Listing Files

This directory contains copy text for the Chrome Web Store submission form.

## Files

- `en.md`
  - English store listing text
  - Paste into:
    - Extension name
    - Short description
    - Detailed description
    - Category
    - Language

- `ja.md`
  - Japanese store listing text
  - Use for the localized Japanese store listing fields if you provide a Japanese translation in the Developer Dashboard

- `justifications.md`
  - English-only review and privacy-practices text
  - Paste into:
    - Single purpose description
    - Permission justifications
    - Host permission justifications
    - Remote code usage answer
    - Privacy practices / data usage declarations
