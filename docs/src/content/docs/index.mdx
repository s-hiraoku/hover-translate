## Why Hover Translate?

A lightweight Chrome extension that translates paragraph-sized text blocks the moment you hover over them. Powered by the DeepL Free API.

Hover Translate is a Chrome extension for reading bilingual pages with less friction. Hover a paragraph, list item, or other text block and the extension shows a translated tooltip in place.

<Image src={popupActive} alt="Hover Translate popup in active state" width={340} />

## How It Works

1. Hover a paragraph or another supported block element on the page.
2. Hover Translate checks whether the text contains Japanese characters and chooses the translation direction automatically.
3. The translated text appears in a floating tooltip near the element you hovered.
