## Why Hover Translate?

A lightweight Chrome extension that translates paragraph-sized text blocks between English and Japanese, on demand, without ever leaving the page. Powered by Chrome's built-in on-device Translator API.

Hover Translate is a Chrome extension for reading bilingual pages with less friction. It ships with two modes: hover a paragraph to translate it, or select the text you want and trigger the translation by shortcut or automatically on release.

<Image src={popupActive} alt="Hover Translate popup in active state" width={340} />

## How It Works

1. Prepare Chrome's English ⇄ Japanese language packs from the extension popup if prompted.
2. Pick a mode: **Hover** translates the nearest paragraph on mouse hover; **Selection** translates the current text selection via shortcut or automatically after you release the mouse.
3. Hover Translate detects the language automatically (Japanese ⇄ English) and shows the result in a floating tooltip anchored to the text.
