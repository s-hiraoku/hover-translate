# ショートカット

Hover Translate のグローバルショートカットは 1 つだけで、Selection モード専用です。

## 既定のショートカット

既定のショートカットは `Alt+Shift+T` です。macOS、Windows、Linux で共通ですが、macOS では `Alt` は `Option` キーに相当するため、`⌥⇧T` と考えると分かりやすい場合があります。

- 拡張機能が有効で、Mode が **Selection**、Trigger が **Shortcut** のときに、**現在の選択範囲**を翻訳します。
- Hover モード、または Selection + Auto モードでは、これらのモードは手動トリガーを必要としないためショートカットは何もしません。

使い方:

1. ポップアップを開いて DeepL API キーを保存し、拡張機能を ON にして、Mode を **Selection**、Trigger を **Shortcut** にします。
2. 対応ページで翻訳したいテキストを選択します。
3. `Alt+Shift+T` を押すと、選択範囲の近くにツールチップで翻訳が表示されます。

## 動作する場所

このショートカットはアクティブタブのコンテンツスクリプトを実行するので、コンテンツスクリプトが注入できるページでのみ動作します。

- ホスト権限の対象となる通常の HTTP/HTTPS サイトで利用できます。
- `chrome://` ページ、Chrome ウェブストア、New Tab ページなどの制限付きページでは**動作しません**。
- これらのページではショートカットを押しても何も起きません。

## ショートカットを変更する

1. `chrome://extensions/shortcuts` を開きます。
2. Hover Translate の **Translate the current text selection** を探します。
3. ショートカット編集欄から、任意のキー組み合わせを設定します。

このページは、日本語 UI では「拡張機能のキーボード ショートカット」と表示されることがあります。ポップアップの **Translate** セクションにある **Change** リンクからも開けます (Mode が Selection + Shortcut のときにのみ表示)。

## ショートカットの競合を避ける

新しいショートカットを保存する前に、次のものと競合しないか確認してください。

- Chrome 本体のショートカット
- OS のショートカット
- 同じキー組み合わせを使う他の拡張機能

競合がある場合、Chrome が設定を拒否したり、タブによって挙動が不安定になったりすることがあります。
