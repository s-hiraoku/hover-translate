# ショートカット

Hover Translate には、ポップアップを開かずに有効・無効を切り替えられるグローバルショートカットが用意されています。

## 既定のショートカット

既定のショートカットは `Alt+Shift+T` です。macOS、Windows、Linux で共通ですが、macOS では `Alt` は `Option` キーに相当するため、`⌥⇧T` と考えると分かりやすい場合があります。

- このショートカットで、拡張機能の ON/OFF を切り替えます。
- 切り替え後、アクティブタブの右上に約 1.2 秒間トーストが表示されます。
- トーストには `Hover Translate: ON` または `Hover Translate: OFF` が表示されます。

## 動作する場所

このショートカットは、拡張機能のコンテンツスクリプトが動作できる通常の Web ページを前提としています。

- ホスト権限の対象となる通常のサイトで利用できます。
- `chrome://` ページや Chrome ウェブストアのような制限付きページでは完全には動作しません。
- これらのページではコンテンツスクリプトが注入されないため、翻訳 UI やトースト表示も行われません。

## ショートカットを変更する

1. `chrome://extensions/shortcuts` を開きます。
2. Hover Translate の **Toggle hover translation on/off** を探します。
3. ショートカット編集欄から、任意のキー組み合わせを設定します。

このページは、日本語 UI では「拡張機能のキーボード ショートカット」と表示されることがあります。ポップアップの **Translate** セクションにある **Change** リンクからも開けます。

## ショートカットの競合を避ける

新しいショートカットを保存する前に、次のものと競合しないか確認してください。

- Chrome 本体のショートカット
- OS のショートカット
- 同じキー組み合わせを使う他の拡張機能

競合がある場合、Chrome が設定を拒否したり、タブによって挙動が不安定になったりすることがあります。
