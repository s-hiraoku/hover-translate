# 設定

ポップアップは **Engine**、**Translate**、**Limits** の 3 セクションです。設定は `chrome.storage.local` の `hoverTranslateState` に保存されます。

## Engine

Chrome built-in Translator API の準備状況を表示します。

- **Ready for English ⇄ Japanese**: 必要な言語パックが利用可能です。
- **Prepare language packs before first use**: 言語パックのダウンロードを開始できますが、ユーザー操作が必要です。
- **Prepare**: 英語 → 日本語、日本語 → 英語の言語パック準備を開始します。
- **Chrome 138+ desktop required**: Translator API が利用できないブラウザです。

## Translate

### ON/OFF トグル

拡張機能全体の有効・無効を切り替えます。OFF のときはどのモードでも翻訳しません。

### Mode

- **Hover**: 300ms の待機後、最も近い段落を翻訳します。
- **Selection**: 選択したテキストだけを翻訳します。

### Trigger

Selection mode では翻訳の発火方法を選べます。

- **Shortcut**: テキスト選択中に `Alt+Shift+T` を押します。
- **Auto**: マウスを離してから 300ms 後に自動翻訳します。

### Shortcut row

現在のショートカットは **Selection + Shortcut** のときだけ表示されます。**Change** は `chrome://extensions/shortcuts` を開きます。

## Limits

- **Max characters per request** は 500 から 5000 まで設定できます。
- 初期値は 1500 です。
- 上限を超えるテキストでは `Text too long` エラーを表示します。
- Hover mode では、翻訳対象の祖先要素を探す範囲の上限にも使われます。

## 保存される状態

```json
{
  "enabled": true,
  "mode": "hover",
  "selectionTrigger": "shortcut",
  "maxChars": 1500
}
```
