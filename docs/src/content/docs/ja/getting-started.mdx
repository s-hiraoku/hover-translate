# はじめに

このガイドでは、Hover Translate のインストール、無料の DeepL API キーの取得、拡張機能が正しく動作することの確認までを説明します。

## 1. 拡張機能をインストールする

1. [Chrome ウェブストアの Hover Translate ページ](https://chromewebstore.google.com/detail/hover-translate-%E8%8B%B1%E8%AA%9E-%E2%87%84-%E6%97%A5%E6%9C%AC%E8%AA%9E/opjkabnjpglbkdlkanjkpkcohmdcajmp?hl=ja) を開きます。
2. **Chrome に追加** をクリックし、確認ダイアログで承認します。
3. インストールが完了すると、Chrome のツールバーに Hover Translate のアイコンが表示されます。

<Aside type="note" title="開発者向けインストール (任意)">
  最新の未リリース変更を試したい、あるいは開発に参加したい場合は、[リポジトリ](https://github.com/s-hiraoku/hover-translate) をクローンし、`pnpm install` と `pnpm build` を実行してから、`chrome://extensions/` の **デベロッパーモード** を有効にして **パッケージ化されていない拡張機能を読み込む** で生成された `dist/` ディレクトリを指定してください。
</Aside>

## 2. 無料の DeepL API キーを取得する

1. [DeepL API plans](https://www.deepl.com/pro-api) にアクセスします。
2. **DeepL API Free** プランに登録します。
3. 登録後、DeepL のアカウント設定を開きます。
4. **API Keys** セクションで API キーを確認し、コピーします。

> 注記: DeepL Free は本人確認のためにクレジットカード登録を求めますが、月 500,000 文字までの利用であれば課金されません。

<Aside type="tip" title="DeepL Free のクォータについて">
  DeepL Free では、月あたり最大 500,000 文字まで無料で利用できます。クォータは課金なしで
  毎月自動的にリセットされ、特別な操作は不要です。上限に達した場合も、次のサイクルまで待てば
  再び利用できます。現在の使用量と次のリセット日は
  [deepl.com/account/usage](https://www.deepl.com/account/usage) で確認できます。
</Aside>

## 3. API キーを接続する

1. Hover Translate の拡張機能アイコンをクリックし、ポップアップを開きます。
2. **Setup** セクションの **DeepL API Key** にキーを貼り付けます。
3. **Save** をクリックします。
4. **Test** をクリックします。
5. ポップアップに `Connection verified.` と表示されれば接続成功です。

<Image src={popupSetup} alt="API key required バナーが表示されたポップアップ設定画面" width={340} />

## 4. 有効化してモードを選ぶ

1. **Translate** セクションで拡張機能を **ON** に切り替えます。
2. モードを選びます。
   - **Hover** — ブロックにホバーすると最寄りの段落を自動翻訳します。
   - **Selection** — 選択した範囲だけを翻訳します。Selection を選ぶと追加のスイッチが表示され、**Shortcut** (`Alt+Shift+T`) と **Auto** (マウス離し検知) を選べます。
3. 英文または日本語文を含む通常の Web ページを開き、選んだモードを試します。
4. 対象テキストの近くに翻訳ツールチップが表示されます。

<Image src={popupActive} alt="Selection モード + Shortcut トリガーが有効なポップアップ画面" width={340} />

ツールチップが表示されない場合やポップアップにエラーが出る場合は、[トラブルシューティング](/hover-translate/ja/troubleshooting/) を参照してください。
