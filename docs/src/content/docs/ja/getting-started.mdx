# はじめに

このガイドでは、Hover Translate のインストールと Chrome built-in Translator API の準備手順を説明します。

## 1. 対応環境を確認する

Hover Translate は Chrome built-in Translator API を使います。

- デスクトップ版 Chrome 138 以降が必要です。
- モバイル Chrome、Firefox、Safari には対応していません。
- 初回利用時に Chrome が英語・日本語の言語パックをダウンロードする場合があります。

<Aside type="note" title="翻訳アカウントは不要です">
  Hover Translate は DeepL などの外部翻訳 API を使いません。API キー、クォータ、課金設定は不要です。
</Aside>

## 2. 拡張機能をインストールする

1. [Chrome Web Store の Hover Translate ページ](https://chromewebstore.google.com/detail/hover-translate-%E8%8B%B1%E8%AA%9E-%E2%87%84-%E6%97%A5%E6%9C%AC%E8%AA%9E/opjkabnjpglbkdlkanjkpkcohmdcajmp) を開きます。
2. **Chrome に追加** をクリックし、確認ダイアログを承認します。
3. インストールが完了すると Chrome のツールバーに Hover Translate のアイコンが表示されます。

<Aside type="note" title="開発者向けインストール">
  ソースから動かす場合は [repository](https://github.com/s-hiraoku/hover-translate)
  を clone し、`pnpm install` と `pnpm build` を実行してから、`chrome://extensions/`
  の **デベロッパー モード** → **パッケージ化されていない拡張機能を読み込む** で `dist/` を読み込みます。
</Aside>

## 3. 言語パックを準備する

1. Hover Translate のアイコンをクリックしてポップアップを開きます。
2. **Engine** に準備が必要と表示された場合は **Prepare** を押します。
3. 英語・日本語の言語パックのダウンロードが終わるまで Chrome を開いたまま待ちます。
4. `Ready for English ⇄ Japanese` と表示されれば準備完了です。

## 4. ON にしてモードを選ぶ

1. **Translate** セクションで拡張機能を ON にします。
2. モードを選びます。
   - **Hover**: 段落にホバーすると自動で翻訳します。
   - **Selection**: 選択したテキストだけを翻訳します。**Shortcut** (`Alt+Shift+T`) または **Auto** を選べます。
3. 英語または日本語のある通常の Web ページで試します。
4. 翻訳結果が元テキスト付近のツールチップに表示されます。

<Image src={popupActive} alt="Selection mode と Shortcut trigger が選択されたポップアップ" width={340} />

表示されない場合は [トラブルシューティング](/hover-translate/ja/troubleshooting/) を確認してください。
