## 特徴

DeepL Free API を使い、ホバーしたテキストブロックを即座に翻訳する軽量 Chrome 拡張機能です。

  <Card title="高速" icon="rocket">

Hover Translate は、英語と日本語が混在するページをより少ない操作で読めるようにする Chrome 拡張機能です。段落やリスト項目などにホバーするだけで、その場に翻訳ツールチップを表示します。

<Image src={popupActive} alt="Hover Translate のポップアップが有効な状態" width={340} />

## 動作の流れ

1. ページ上の段落や対応するブロック要素にホバーします。
2. Hover Translate がテキスト内に日本語文字が含まれるかどうかを確認し、翻訳方向を自動で選びます。
3. 翻訳結果が、ホバーした要素の近くにフローティングツールチップとして表示されます。
