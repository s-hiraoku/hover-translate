## 特徴

DeepL Free API を使い、ページ上の英語⇄日本語テキストをその場で翻訳する軽量 Chrome 拡張機能です。

<CardGrid>
  <Card title="2つのモード" icon="setting">
    段落にホバーして自動翻訳する Hover モードと、選択範囲だけを翻訳する Selection モードを切り替えられます。
  </Card>
  <Card title="自動言語判定" icon="translate">

Hover Translate は、英語と日本語が混在するページをより少ない操作で読めるようにする Chrome 拡張機能です。ホバーで段落を自動翻訳する Hover モードと、選択したテキストだけをショートカットまたはマウス離し検知で翻訳する Selection モードの 2 つを備えています。

<Image src={popupActive} alt="Hover Translate のポップアップが有効な状態" width={340} />

## 動作の流れ

1. ポップアップから DeepL Free API キーを保存します。
2. モードを選びます。**Hover** は対象の段落にホバーすると自動で翻訳し、**Selection** は選択したテキストをショートカットまたはマウスアップ後に自動翻訳します。
3. Hover Translate がテキストの言語 (日本語 ⇄ 英語) を自動判定し、翻訳結果をテキストの近くにフローティングツールチップで表示します。
